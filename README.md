# Wearable Rehabilitation Motion Tracker

## Project Overview

The Wearable Rehabilitation Motion Tracker is a prototype wearable device designed to monitor rehabilitation exercises using motion sensing and embedded processing.

The system utilizes an ESP32 microcontroller, BNO055 inertial measurement unit (IMU), Bluetooth Low Energy (BLE) communication, and a rechargeable power system to analyze movement and provide rehabilitation-focused feedback.

---

## Problem Statement

Rehabilitation patients often perform exercises without direct supervision, making it difficult to monitor exercise quality, range of motion, and exercise adherence.

This project explores how wearable motion tracking can provide objective rehabilitation metrics and support recovery-focused feedback.
